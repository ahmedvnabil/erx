# Go-to-Market — ERX

## التموضع

**ERX هو طبقة المصدر المفتوحة للشأن المصري داخل البحث البشري ووكلاء AI.**

الفئة الجديدة ليست «محرك أخبار» ولا «شات بوت مصري». الفئة هي Research Infrastructure: أرشيف موحد، مقارنة مصادر، استشهادات، وMCP يمكن لأي عميل استخدامه.

## نقطة الدخول

نبدأ بالباحثين التقنيين والصحفيين الذين يستخدمون Claude وChatGPT وCursor؛ لأنهم يفهمون قيمة MCP ويمكنهم إثبات حالات استخدام واضحة. بعد ذلك ننتقل إلى مراكز الدراسات والجامعات عبر واجهة البحث، دون اشتراط فهم MCP.

## قنوات التوزيع

1. GitHub public repo مع demo مرئي وطلبات إضافة مصدر.
2. npm لسهولة `npx` والتثبيت.
3. Official MCP Registry بوصفه المصدر القانوني لاكتشاف الخادم.
4. Glama لتجربة الأدوات من المتصفح وفهرسة schemas.
5. PulseMCP وأدلة MCP المجتمعية.
6. X وLinkedIn لحالات الاستخدام والنتائج، لا لإعلانات عامة مبهمة.
7. مجموعات الصحافة البيانية وArabic NLP والجامعات ومراكز السياسات.
8. رسائل تعريفية قصيرة لمؤسسات لديها مكتبات أو بيانات مصرية قابلة للإضافة.

## الحلقة التسويقية

```text
بحث أو مثال موثق
  → لقطة نتيجة مع رابط المصدر
  → رابط Landing أو مستودع GitHub
  → تجربة البحث أو نسخ أمر npx
  → طلب مصدر جديد / Star / Integration
  → اتساع الكتالوج وظهور أمثلة أقوى
```

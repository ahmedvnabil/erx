# المساهمة

نرحب بإضافة مصادر مصرية موثوقة، وتحسين الاستخراج، وتصحيح البيانات الوصفية.

1. افتح issue يوضح المصدر، نوعه، مالكه، ورابط قناة الجمع المباشرة إن وجدت.
2. لا تضف مفاتيح API أو بيانات دخول أو محتوى خلف جدار اشتراك.
3. أضف اختبارًا قبل التغيير وشغّل مجموعة الاختبارات كاملة.
4. لا تمنح مصدرًا درجة حقيقة ولا تحذف رابط الأصل من النتائج.

```bash
npm ci
npm run check
npm run test:coverage
npm run build
```

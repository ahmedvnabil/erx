# Security

أرسل بلاغات الثغرات بصورة خاصة إلى `vacrom414@gmail.com`، ولا تنشر تفاصيل
قابلة للاستغلال قبل إتاحة الإصلاح.

واجهات القراءة العامة محددة المعدل داخل العملية، وتضيف الخدمة CSP ورؤوس حماية
ومعرّفًا مستقلًا لكل طلب. أداة حفظ البحث تكتب بيانات محلية غير حساسة فقط.

استخدم HTTPS وبوابة موثوقة قبل تعريض الخدمة للإنترنت العام. عند تشغيل أكثر من
نسخة، ضع تحديد معدل موزع أمامها، ولا تجعل `/metrics` متاحًا للعامة. لا تسجل
`GEMINI_API_KEY` أو تضعه في عنوان URL؛ يُمرر عبر متغير البيئة فقط.

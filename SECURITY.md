# Security Policy

## Supported versions

يتم دعم أحدث إصدار منشور فقط حتى يصل المشروع إلى `1.0.0`.

## Reporting a vulnerability

لا تفتح Issue عامة تتضمن تفاصيل استغلال أو أسرارًا. أرسل وصفًا مختصرًا إلى `vacrom414@gmail.com` يتضمن النسخة المتأثرة، خطوات إعادة الإنتاج، والأثر المتوقع. سيصل تأكيد الاستلام خلال 72 ساعة قدر الإمكان.

## Scope

يشمل النطاق SSRF في جامعات المصادر، تجاوز حدود الملفات، injection في البحث أو HTML، تسريب مفاتيح embeddings، وتجاوز rate limits بشكل يؤثر في الخدمة أو البيانات.

المحتوى الخاطئ الصادر من مصدر خارجي ليس ثغرة في حد ذاته، لكن فقدان الاستشهاد أو نسب المحتوى إلى مصدر غير صحيح يُعامل كخلل سلامة بيانات عالي الأولوية.

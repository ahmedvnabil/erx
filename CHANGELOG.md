# Changelog

## 0.3.0 — 2026-07-14

- جمع متعدد القنوات من RSS وSitemap وHTML وPDF مع OCR عربي واحترام robots.txt.
- طبقة معرفة للكيانات والأحداث والادعاءات مع إرجاع الأدلة إلى الوثيقة الأصلية.
- بحث هجين نصي ودلالي محلي، ودعم اختياري لـ Gemini Embeddings.
- خمس أدوات MCP جديدة وREST API إصدار v1 وصفحة خريطة معرفة عربية.
- رؤوس أمان، تحديد معدل، سجلات طلبات منظمة، readiness ومقاييس Prometheus.
- ترحيلات بيانات، نسخ SQLite متسق، تحقق سلامة، استعادة آمنة، وفهرسة backfill.
- صورة إنتاج تدعم PDF/OCR، CI للحاوية والتبعيات، وحزمة إصدار عند الوسم.

## 0.2.0 — 2026-07-14

- إضافة واجهة بحث عربية RTL تعمل مع MCP في الخدمة نفسها.
- استخراج اختياري لمتن المقال الكامل مع قيود أمان.
- تجميع القصص المتشابهة داخل نافذة زمنية محددة.
- سجل عمليات الجمع وحالة المصادر.
- تصدير CSV وJSONL وBibTeX وRIS.
- حاوية Docker، فحص صحة، وملفات مساهمة وأمان واستشهاد.

## 0.1.0 — 2026-07-14

- إطلاق كتالوج 33 مصدرًا، أرشيف SQLite FTS5، وسبع أدوات MCP أساسية.

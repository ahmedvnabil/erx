# Changelog

## 0.7.0 — 2026-07-15

- إضافة موصلات HTML مخصصة لمجلس النواب ومجلس الوزراء والمجلس القومي لحقوق الإنسان والمنصة ومؤسسة سيناء ومراسلون بلا حدود.
- إضافة موصلات API رسمية للجهاز المركزي للتعبئة العامة والإحصاء وأحكام المحكمة الدستورية العليا.
- إضافة واجهة IDSC الرسمية، بما في ذلك طلب POST وبنية الاستجابة المتداخلة.
- إضافة HTML احتياطي للمبادرة المصرية وحرية الفكر والتعبير والمصري اليوم وأخبار اليوم ومصراوي واليوم السابع والشروق عند تعطل RSS.
- توسيع تدقيق المصادر ليفحص روابط المقالات وسجلات JSON، لا مجرد استجابة صفحة المصدر.
- معالجة HTML غير المغلق في موقع مجلس النواب دون فقد متن الخبر.
- فصل حد صفحات Sitemap عن حد مقالات HTML في مهمة الجمع الدورية.
- توضيح الفرق في صفحة المصادر بين «يعمل» و«متعطل» و«كتالوج فقط» بدل عرض الحالات التقنية الخام.

## 0.6.0 — 2026-07-15

- رفع قنوات الجمع المتحققة من 8 إلى 19 مصدرًا: 18 RSS وSitemap واحدة.
- إضافة `audit-sources` لفحص المصادر الـ33 من بيئة التشغيل الفعلية وتصنيف نوع منصة الموقع.
- التحقق من بنية RSS وSitemap وعدد العناصر بدل الاكتفاء بحالة HTTP.
- فصل الحجب وتحديد المعدل والتحويلات غير المتوافقة والمحتوى غير الصالح وأخطاء الشبكة.
- منع عناصر RSS التي تشير إلى نطاق خارج نطاق المصدر الموثق.

## 0.5.1 — 2026-07-15

- إكمال Landing Page ثنائية اللغة بواجهة بصرية للبراند وإحصاءات حية ومساري اتصال محلي وعام.
- حماية Streamable HTTP عبر التحقق من Origin وإرجاع أخطاء JSON-RPC صحيحة للطلبات غير الصالحة.
- تعطيل الكتابة المشتركة في نقطة MCP العامة مع الإبقاء على الحفظ المحلي عبر stdio.
- تصحيح تحديد المعدل خلف Nginx لكل عميل مع مقاومة تزوير `X-Forwarded-For`.
- تمييز حجب المصدر بـ403 عن أخطاء الجلب والتحليل العامة داخل سجل الجمع.
- إضافة دعم HEAD للصفحة الرئيسية وتوحيد رقم الإصدار بين الحزمة والسيرفر والاستشهاد والكود.
